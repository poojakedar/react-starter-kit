# React Starter Kit ⚛️

A modern and lightweight React starter kit powered by Vite, designed to help you kickstart development quickly with best practices.

---

## 🚀 Features

- ⚡ Vite for fast development
- ✅ ESLint configured
- 🎯 Prettier for code formatting
- 📁 Clean and scalable folder structure
- 🔧 Ready-to-use scripts
